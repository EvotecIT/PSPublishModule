# Add-BenchmarkComparison
## SYNOPSIS
Adds a benchmark comparison definition.

## SYNTAX
### __AllParameterSets
```powershell
Add-BenchmarkComparison [[-Dimension] <string>] -Baseline <string> [-Metric <string[]>] [-TieTolerance <double>] [<CommonParameters>]
```

## DESCRIPTION
Adds a benchmark comparison definition.

## EXAMPLES

### EXAMPLE 1
```powershell
Add-BenchmarkComparison -Baseline 'Value'
```
