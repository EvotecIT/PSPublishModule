# Test-AppStoreConnectReleaseReadiness
## SYNOPSIS
Checks whether an App Store Connect Distribution version is ready for submission.

## SYNTAX
### __AllParameterSets
```powershell
Test-AppStoreConnectReleaseReadiness -IssuerId <string> -KeyId <string> -AppId <string> -VersionString <string> -Platform <ApplePlatform> [-PrivateKey <string>] [-PrivateKeyPath <string>] [-TokenLifetimeMinutes <int>] [-BuildNumber <string>] [-Locale <string>] [-RequiredScreenshotDisplayTypes <string[]>] [-MinimumScreenshotsPerSet <int>] [-NoRequireSelectedBuild] [-NoRequireValidBuild] [-NoRequireDescription] [-NoRequireKeywords] [-NoRequireSupportUrl] [-RequireMarketingUrl] [-RequirePromotionalText] [-RequireWhatsNew] [-NoRequireScreenshots] [-NoRequireCompleteScreenshots] [<CommonParameters>]
```

## DESCRIPTION
Checks whether an App Store Connect Distribution version is ready for submission.

## EXAMPLES

### EXAMPLE 1
```powershell
Test-AppStoreConnectReleaseReadiness -IssuerId 'Value' -KeyId 'Value' -AppId 'Value' -VersionString 'Value' -Platform 'Value'
```
