# New-AppStoreConnectBetaTester
## SYNOPSIS
Creates an App Store Connect TestFlight beta tester.

## SYNTAX
### __AllParameterSets
```powershell
New-AppStoreConnectBetaTester -IssuerId <string> -KeyId <string> -Email <string> [-PrivateKey <string>] [-PrivateKeyPath <string>] [-TokenLifetimeMinutes <int>] [-FirstName <string>] [-LastName <string>] [-BetaGroupId <string[]>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Creates an App Store Connect TestFlight beta tester.

## EXAMPLES

### EXAMPLE 1
```powershell
New-AppStoreConnectBetaTester -IssuerId 'Value' -KeyId 'Value' -Email 'Value'
```
