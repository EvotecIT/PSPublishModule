# Invoke-PowerForgeBundlePostProcess
## SYNOPSIS
Applies reusable bundle post-process rules from a dotnet publish config to an existing bundle directory.

```powershell
Invoke-PowerForgeBundlePostProcess
```

## DESCRIPTION
Applies reusable bundle post-process rules from a dotnet publish config to an existing bundle directory.
