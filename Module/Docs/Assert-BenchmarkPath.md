# Assert-BenchmarkPath
## SYNOPSIS
Asserts that a benchmark path exists or does not exist.

## SYNTAX
### __AllParameterSets
```powershell
Assert-BenchmarkPath [-Path] <string> [-Not] [-Message <string>] [-PassThru] [<CommonParameters>]
```

## DESCRIPTION
Asserts that a benchmark path exists or does not exist.

## EXAMPLES

### EXAMPLE 1
```powershell
Assert-BenchmarkPath -Path 'C:\Path'
```
