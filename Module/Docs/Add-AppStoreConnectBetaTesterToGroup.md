# Add-AppStoreConnectBetaTesterToGroup
## SYNOPSIS
Adds App Store Connect beta testers to a TestFlight beta group.

## SYNTAX
### __AllParameterSets
```powershell
Add-AppStoreConnectBetaTesterToGroup -IssuerId <string> -KeyId <string> -BetaGroupId <string> -BetaTesterId <string[]> [-PrivateKey <string>] [-PrivateKeyPath <string>] [-TokenLifetimeMinutes <int>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Adds App Store Connect beta testers to a TestFlight beta group.

## EXAMPLES

### EXAMPLE 1
```powershell
Add-AppStoreConnectBetaTesterToGroup -IssuerId 'Value' -KeyId 'Value' -BetaGroupId 'Value' -BetaTesterId @('Value')
```
