# New-AppStoreConnectScreenshotSet
## SYNOPSIS
Creates an App Store Connect screenshot set for an App Store version localization.

## SYNTAX
### __AllParameterSets
```powershell
New-AppStoreConnectScreenshotSet -IssuerId <string> -KeyId <string> -VersionLocalizationId <string> -ScreenshotDisplayType <string> [-PrivateKey <string>] [-PrivateKeyPath <string>] [-TokenLifetimeMinutes <int>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Creates an App Store Connect screenshot set for an App Store version localization.

## EXAMPLES

### EXAMPLE 1
```powershell
New-AppStoreConnectScreenshotSet -IssuerId 'Value' -KeyId 'Value' -VersionLocalizationId 'Value' -ScreenshotDisplayType 'Value'
```
