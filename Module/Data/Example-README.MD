# PowerShell Module

This module was scaffolded with `PSPublishModule`.

## Build

```powershell
.\Build\Build-Module.ps1
```

Use `-ConfigurationGateMode Publish` only when you want publish-gated settings to run.

## Documentation

`Docs\` and `en-US\*-help.xml` are generated by the build. Author command help in source and about topics under `Help\About\about_*.help.txt`.

```powershell
New-ModuleAboutTopic -TopicName 'Overview' -OutputPath '.\Help\About'
```
