# Initialize-ProjectManager

## SYNOPSIS
Builds VSCode Project manager config from filesystem

## SYNTAX

```
Initialize-ProjectManager [-Path] <String> [<CommonParameters>]
```

## DESCRIPTION
Builds VSCode Project manager config from filesystem

## EXAMPLES

### EXAMPLE 1
```
Initialize-ProjectManager -Path "C:\Support\GitHub"
```
