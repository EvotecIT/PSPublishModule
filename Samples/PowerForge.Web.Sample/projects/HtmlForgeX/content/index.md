# HtmlForgeX

HtmlForgeX provides a typed builder model for generating HTML dashboards, reports, and exports with consistent styling.

## Highlights

- Typed components for layout, cards, tables, and charts.
- Build HTML reports from PowerShell and .NET.
- Theme-ready with reusable styles and snippets.
